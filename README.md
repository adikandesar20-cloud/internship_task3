# internship_task3
About This repository contains source code for my internship task 3.
Even Odd Checker

# Objective
To develop a Java program that checks whether a number is even or odd.

# Concepts Used
- Modulo Operator (%)
- Ternary Operator (? :)
- User Input using Scanner
- Methods (Function)

# Description
This program takes an integer input from the user and determines whether it is even or odd.  
It uses the modulo operator to check divisibility by 2 and a method to display the result.
